# Keyboard Controls Quick Reference

## Navigation
```
↑ = Parent Node        ↓ = Child Node
← = Left Sibling       → = Right Sibling
```

## Selection
```
Enter/Space = Select        Esc = Deselect
```

## Tree Control
```
+ = Expand Subtree     - = Collapse Subtree
```

## Quick Actions
```
E = Edit Selected      A = Add Relative
```

## In TreeViewer Help Sidebar
All keyboard shortcuts are displayed in the "How to Use" sidebar under the "Keyboard Shortcuts" section.

## Notes
- Arrow key directions adapt to tree layout orientation (vertical vs horizontal)
- E and A keys require a selected person first
- Plus/Minus keys only work on parent nodes
- Keyboard input is ignored while typing in forms
